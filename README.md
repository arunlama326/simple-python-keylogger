# Simple Python Keylogger

An educational cybersecurity tool that captures keystrokes and saves them to a human-readable `logs.txt` file.

## ⚠️ Disclaimer
This project is for **educational and ethical security testing purposes only**. Using this software against targets without prior informed consent is illegal.

## Installation
1. Clone the repo: `git clone https://github.com/arunlama326/simple-python-keylogger.git`
2. Install dependencies: `pip install -r requirements.txt`

## Usage
Run the script with root privileges (required for keyboard listeners):
```bash
sudo python keylogger.py
